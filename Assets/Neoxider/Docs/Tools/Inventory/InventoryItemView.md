# InventoryItemView

**Что это:** компонент одной строки/карточки предмета в UI инвентаря: иконка, название, количество (поля опциональны). Используется внутри [InventoryView](./InventoryView.md) в режимах SpawnFromPrefab или ManualList. Пространство имён: `Neo.Tools`. Файл: `Scripts/Tools/Inventory/UI/InventoryItemView.cs`.

**Как использовать:** добавить на префаб элемента списка или на дочерний объект; при SpawnFromPrefab/ManualList InventoryView заполняет данные из InventoryComponent. При необходимости подписаться на клик/события компонента.
