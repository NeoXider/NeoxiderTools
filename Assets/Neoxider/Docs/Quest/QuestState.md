# QuestState

**Назначение:** Класс, хранящий текущее состояние квеста (прогресс) во время игры. `QuestState` генерируется `QuestManager`ом, когда игрок берет квест, и хранится в памяти, а также сериализуется в JSON для сохранений.

## Описание

`QuestState` не настраивается в Инспекторе напрямую (за исключением дебага, если вы смотрите массив внутри QuestManager). Это чисто рантайм-объект.

Он хранит:
1. Идентификатор квеста (`_questId`).
2. Текущий статус (`_status`: NotStarted, Active, Completed, Failed).
3. Список текущего прогресса по каждому заданию (сколько кабанов убито).
4. Список флагов завершения по каждому заданию (`true`/`false`).

## Использование в коде

Обычно вам не нужно вручную менять `QuestState`. Вы взаимодействуете с ним через `QuestManager`.
Однако, если вы хотите отобразить прогресс в UI, вы можете получить его:

```csharp
// Получить состояние активного квеста
QuestState state = QuestManager.I.GetState("kill_boars");

if (state != null) {
    // Узнать, сколько мы уже собрали предметов по первой задаче (индекс 0)
    int currentAmount = state.GetObjectiveProgress(0);
    
    // Проверить, завершена ли задача
    bool isDone = state.IsObjectiveCompleted(0);
}
```
