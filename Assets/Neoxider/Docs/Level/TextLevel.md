# TextLevel

**Что это:** `TextLevel` — компонент отображения уровня, который наследуется от `SetText` и подписывается на `LevelManager`, чтобы выводить текущий или максимальный уровень. Файл: `Scripts/Level/TextLevel.cs`, пространство имён: `Neo.Level`.

**Как использовать:**
1. Добавьте `TextLevel` на объект с текстовым выводом.
2. При необходимости задайте `Level Source`, если на сцене больше одного `LevelManager`.
3. Выберите режим отображения: текущий уровень или максимальный.
4. Настройте `Display Offset`, если пользователю нужно показывать уровни начиная с `1`, а не с `0`.

---

## Основные настройки

- `Level Source` — конкретный `LevelManager`. Если не задан, используется `LevelManager.I`.
- `Display Mode` — что показывать: текущий уровень или максимум.
- `Display Offset` — сдвиг отображаемого значения.

## Как работает

- В `Start()` компонент пытается найти `LevelManager`.
- Если менеджер ещё не доступен, используется отложенная инициализация.
- В режиме `Current` подписывается на `LevelManager.OnChangeLevel`.
- В режиме `Max` подписывается на `LevelManager.OnChangeMaxLevel`.
- При `OnDisable()` снимает подписку.

## Особенности

- Старый скрытый флаг `_best` по сути переводит компонент в режим `Max`.
- Для сцен с несколькими `LevelManager` лучше всегда задавать `Level Source` явно.
