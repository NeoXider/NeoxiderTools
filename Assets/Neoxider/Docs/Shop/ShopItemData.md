# ShopItemData

**Назначение:** `ScriptableObject` для хранения информации о товаре в магазине. Позволяет настраивать свойства товара (цена, иконка, название) прямо в инспекторе без изменения кода.

## Подключение

1. Создайте объект данных товара через контекстное меню: `Right Click > Create > Neoxider > Shop > Shop Item Data`.
2. Настройте поля.
3. Добавьте созданный объект в массив `_shopItemDatas` контроллера `Shop`.

## Основные настройки (Inspector)

| Поле | Описание |
|------|----------|
| `_isSinglePurchase` | Может ли этот предмет быть куплен только один раз? (Например, уникальный скин или уровень). |
| `_nameItem` | Название товара, отображаемое в магазине. |
| `_description` | Описание товара. |
| `_price` | Начальная цена товара. |
| `_sprite` | Главная картинка товара (например, для окна предпросмотра). |
| `_icon` | Иконка товара (например, для маленькой ячейки в списке). |
