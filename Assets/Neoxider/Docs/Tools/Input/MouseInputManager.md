# MouseInputManager

Синглтон ввода мыши без аллокаций: Press, Hold, Release, Click с данными (MouseEventData). Raycast 2D/3D, опциональная блокировка UI.

**Добавить:** в сцену один раз (синглтон).

## Основное

- **LastEventData**, **HasEventData** — последнее событие для опроса.
- Подписка через делегаты/события на нажатие, удержание, отпускание, клик.

Используется для единой точки ввода мыши в игре (меню, тапы по полю).
