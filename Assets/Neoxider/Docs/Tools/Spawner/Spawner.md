# Spawner

**Назначение:** Гибкий компонент для спавна объектов. Поддерживает работу с пулами (`PoolManager`), бесконечный спавн, спавн волнами (Waves), случайную задержку, случайный поворот и спавн внутри коллайдера (2D/3D).

## Поля (Inspector)

| Поле | Описание |
|------|----------|
| **Prefabs** | Массив префабов. При спавне случайным образом выбирается один из них. |
| **Use Object Pool** | Использовать ли `PoolManager` (если он есть в сцене) вместо обычного `Instantiate`. |
| **Destroy Delay** | Время в секундах до автоматического уничтожения (деспавна) объекта. `0` = не уничтожать. |
| **Spawn Limit** | Лимит спавна в режиме `Loop`. `0` = бесконечно. |
| **Min / Max Spawn Delay** | Минимальная и максимальная задержка между спавном соседних объектов (в секундах). |
| **Spawn Mode** | `Loop` (классический непрерывный спавн) или `Waves` (волны с паузами). |
| **Base Wave Count** | Количество объектов в первой волне. |
| **Count Per Wave** | На сколько объектов увеличивать каждую следующую волну. |
| **Time Between Waves** | Пауза между волнами (в секундах). |

## API

| Метод / Свойство | Описание |
|------------------|----------|
| `void StartSpawn()` | Запускает процесс автоматического спавна (волны или цикл). |
| `void StopSpawn()` | Останавливает автоматический спавн. Управляет флагом `isSpawning`. |
| `GameObject SpawnRandomObject()` | Разово спавнит случайный префаб из списка прямо сейчас и возвращает его. |
| `GameObject SpawnById(int prefabId, Vector3 position)` | Спавнит конкретный префаб из массива по индексу в указанной позиции. |
| `void Clear()` | Останавливает спавн и уничтожает (или возвращает в пул) все ранее заспавненные этим спавнером объекты. |

## Unity Events

| Событие | Аргументы | Описание |
|---------|-----------|----------|
| `OnObjectSpawned` | `GameObject` | Вызывается каждый раз, когда объект успешно заспавнен. |
| `OnWaveStarted` | `int waveIndex` | Вызывается при старте новой волны (возвращает номер волны). |
| `OnWaveObjectSpawned` | `GameObject, int` | Вызывается при спавне объекта внутри волны (возвращает сам объект и номер волны). |

## Примеры

### Пример No-Code (в Inspector)
Настройте компонент на режим `Waves`, задав `Base Wave Count = 3` и `Count Per Wave = 2`. Установите `Spawn On Awake`. На событие `OnWaveStarted` повесьте метод UI-скрипта, который будет выводить текст "Началась волна N!" на экран.

### Пример (Код)
```csharp
[SerializeField] private Spawner _enemySpawner;

public void StartBossFight()
{
    // Разово спавним первого врага в центре комнаты
    _enemySpawner.SpawnById(0, Vector3.zero);
    
    // Запускаем бесконечный спавн подмоги
    _enemySpawner.StartSpawn();
}
```
