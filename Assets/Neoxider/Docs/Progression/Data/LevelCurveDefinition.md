# LevelCurveDefinition

**Назначение:** ScriptableObject — кривая уровней (формула XP для каждого уровня).

## Подключение

- Создать: **Create → Neoxider → Level → Level Curve**.

## Основные настройки (Inspector)

| Поле | Описание |
|------|----------|
| `1` | 1. |
| `GrantedPerkPoints` | Granted Perk Points. |
| `Level` | Level. |
| `Levels` | Levels. |
| `RequiredXp` | Required Xp. |
| `Rewards` | Rewards. |
| `_grantedPerkPoints` | Granted Perk Points. |
| `_levels` | Levels. |
| `_requiredXp` | Required Xp. |
| `_rewards` | Rewards. |
