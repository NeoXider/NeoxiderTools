# Tools / Components

Reusable components: counters, Animator driver, score, typewriter, loot, lifecycle events, and more.

## Main components

- **[AnimatorParameterDriver](AnimatorParameterDriver.md)** — Drive Animator parameters (trigger, bool, float, int) from code or UnityEvent.
- **[Counter](Counter.md)** — Universal counter (int/float), Add/Subtract/Multiply/Divide/Set, optional save, OnValueChanged / OnSend events.
- **[ScoreManager](ScoreManager.md)** — Score source for UI and other components.
- **[TypewriterEffect](TypewriterEffect.md)** — Typewriter text effect.
- **[Loot](Loot.md)** — Loot/drop logic.
- **[UnityLifecycleEvents](UnityLifecycleEvents.md)** — Expose Unity lifecycle as UnityEvents.

## Submodules

- **[AttackSystem](AttackSystem/README.md)** — Health, Evade, AttackExecution, AdvancedAttackCollider.
- **[Interface](Interface/README.md)** — InterfaceAttack and related.
