# Local (repo root)

Содержимое подкаталога **`Audits/`** в git **не отслеживается** (см. `.gitignore`).

Сюда переносите **временные** отчёты и аудиты, чтобы не коммитить их в репозиторий, например:

- `FULL_AUDIT.md`
- `NO_CODE_AUDIT.md`
- `COVERAGE_AUDIT.md`

Пути внутри этих файлов на документацию пакета настроены относительно `Local/Audits/` → `Assets/Neoxider/Docs/...`.
