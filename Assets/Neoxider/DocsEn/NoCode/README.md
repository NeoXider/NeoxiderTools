# Neo.NoCode

## Purpose

No-code bindings for UI without writing glue scripts: text, formatted text, and progress visuals from `ReactiveProperty*` / field sources.

## What is documented

- `NoCodeBindText`
- `NoCodeFormattedText`
- `SetProgress`
- Source resolution strategies and fallback behavior
