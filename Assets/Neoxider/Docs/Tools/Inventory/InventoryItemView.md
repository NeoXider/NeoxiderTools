# InventoryItemView

Представление одной записи предмета в UI инвентаря: иконка, название, количество (все поля опциональны). Используется внутри [InventoryView](./InventoryView.md) в режимах SpawnFromPrefab или ManualList.

- **Пространство имён:** `Neo.Tools`
- **Путь:** `Assets/Neoxider/Scripts/Tools/Inventory/UI/InventoryItemView.cs`

См. также [InventoryView](./InventoryView.md), [InventoryComponent](./InventoryComponent.md).
