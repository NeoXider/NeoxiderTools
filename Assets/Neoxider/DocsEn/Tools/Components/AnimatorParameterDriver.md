# AnimatorParameterDriver

**Purpose:** See Inspector fields below for configuration.

## Setup

- Add the component via the Unity menu.

## Key Fields (Inspector)

| Field | Description |
|-------|-------------|
| `animator` | Animator. |
| `boolParameterName` | Bool Parameter Name. |
| `floatParameterName` | Float Parameter Name. |
| `intParameterName` | Int Parameter Name. |
| `triggerParameterName` | Trigger Parameter Name. |
