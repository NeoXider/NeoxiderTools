# PlayAudioBtn

**Что это:** компонент для воспроизведения UI-звука по выбранному событию указателя/навигации (пространство имён `Neo.Audio`, файл `Scripts/Audio/AudioSimple/PlayAudioBtn.cs`). Поддерживает режимы: по ID из [AM](AM.md) (legacy) или из списка `AudioClip` с опцией случайного выбора.

**Как использовать:** добавить на UI-элемент с raycast/selection событиями. Выбрать `Trigger Mode` (`PointerClick`, `PointerEnter`, `PointerExit`, `PointerDown`, `PointerUp`, `Select`, `Deselect`, `Submit`, `Manual`). Задать ID или список клипов в Inspector. При `Manual` звук не запускается автоматически, только через `AudioPlay()` из кода или UnityEvent.

---

## Поля и методы

**Ключевые особенности**
- **Двойной режим работы**: Поддерживает как воспроизведение по ID (legacy), так и напрямую по `AudioClip`.
- **Случайный выбор**: Если задан список клипов и включен `useRandomClip`, выбирается случайный клип из списка.
- **Гибкий trigger mode**: можно проигрывать звук не только по клику, но и по наведению, отводу курсора, нажатию, отпусканию, выбору через навигацию и submit.
- **Manual режим**: удобно для вызова из собственного кода или UnityEvent.

**Поля в инспекторе**
- `_idClip` (`int`): ID звука в массиве AM (legacy режим).
- `_clips` (`AudioClip[]`): Список клипов для воспроизведения (новый режим).
- `_triggerMode` (`TriggerMode`): событие, по которому запускать звук.
- `_useRandomClip` (`bool`): Если `true` и в списке больше 1 клипа - выбирается случайный.
- `_volume` (`float`): Громкость воспроизведения (0-1).

**TriggerMode**
- `PointerClick` — левый клик мышью / тап.
- `PointerEnter` — наведение курсора.
- `PointerExit` — уход курсора.
- `PointerDown` — нажатие кнопки мыши / тапа.
- `PointerUp` — отпускание кнопки мыши / тапа.
- `Select` — выбор UI-элемента через EventSystem/навигацию.
- `Deselect` — снятие выбора.
- `Submit` — подтверждение через UI navigation submit.
- `Manual` — только ручной вызов `AudioPlay()`.

**Публичные методы**
- `AudioPlay()`: Воспроизводит настроенный звуковой эффект. Если заданы клипы - использует их (с случайным выбором если включено), иначе использует режим по ID.
