# GridSystem

Field generators, cell state, pathfinding, shape/origin config, Match3 and TicTacToe game layers. Scripts in `Scripts/GridSystem/`. Full architecture, API and examples are in the Russian single-file doc.

## Samples

GridSystem demos live in the package UPM sample `Demo Scenes`:

- `Assets/Neoxider/Samples~/Demo/Scenes/GridSystem/GridSystemMatch3Demo.unity`
- `Assets/Neoxider/Samples~/Demo/Scenes/GridSystem/GridSystemTicTacToeDemo.unity`
- `Assets/Neoxider/Samples~/Demo/Scripts/GridSystem/GridSystemMatch3DemoSetup.cs`
- `Assets/Neoxider/Samples~/Demo/Scripts/GridSystem/GridSystemTicTacToeDemoSetup.cs`
- `Assets/Neoxider/Samples~/Demo/Scripts/GridSystem/GridSystemMatch3BoardView.cs`
- `Assets/Neoxider/Samples~/Demo/Scripts/GridSystem/GridSystemTicTacToeBoardView.cs`

Local sample-code verification uses `Neo.Rpg.Demo.csproj`; its include paths should stay under `Assets/Neoxider/Samples~/Demo/...`.
