# InventoryItemCountText

Текстовое поле, отображающее количество конкретного предмета в инвентаре. Привязывается к `InventoryComponent` и заданному `Item Id`; обновляется при изменении инвентаря.

- **Пространство имён:** `Neo.Tools`
- **Путь:** `Assets/Neoxider/Scripts/Tools/Inventory/UI/InventoryItemCountText.cs`

См. также [InventoryComponent](./InventoryComponent.md), [InventoryView](./InventoryView.md).
