# Map

**Что это:** `Map` — сериализуемый класс данных внутри модуля `Level`, который хранит прогресс одной карты: количество уровней, loop/infinity-режим и текущий открытый уровень. Файл: `Scripts/Level/Map.cs`, пространство имён: `Neo.Level`.

**Как использовать:**
1. Настраивайте `Map` как элемент массива `Maps` в `LevelManager`.
2. Используйте `Load()` только через `LevelManager`, если не строите свою систему вручную.
3. Для фиксации прогресса вызывайте `SaveLevel()` через `LevelManager.SaveLevel()` или напрямую, если понимаете текущую схему сохранения.

---

## Поля

| Поле | Описание |
|------|----------|
| `isInfinity` | Если `true`, карта считается бесконечной и не имеет обычного completion-сценария. |
| `countLevels` | Количество уровней в карте. |
| `isLoopLevel` | Если `true`, уровни циклически повторяются через `GetLoopLevel()`. |
| `idMap` | Внутренний индекс карты в массиве `LevelManager`. |
| `level` | Текущий открытый уровень карты. |

## Основные методы

| Метод | Описание |
|-------|----------|
| `Load(int i, string saveKey)` | Загружает прогресс карты через `SaveProvider`. |
| `SaveLevel()` | Увеличивает и сохраняет текущий уровень карты. |
| `GetCopmplete()` | Возвращает `true`, если карта завершена и не является infinity-картой. |

## Сохранение

Ключ строится в формате:

```text
Map_{saveKey}_{idMap}_level
```

То есть каждая карта хранит собственный прогресс отдельно внутри общего ключа `LevelManager`.
