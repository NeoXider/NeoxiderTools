# ParallaxLayer

**Purpose:** See Inspector fields below for configuration.

## Setup

- Add the component via the Unity menu.

| Field | Description |

| `parallaxMultiplier` | Parallax Multiplier. |
| `randomiseOnRecycle` | Randomise On Recycle. |
| `scrollSpeed` | Scroll Speed. |
| `spriteVariants` | Sprite Variants. |
| `targetCamera` | Target Camera. |
| `templateRenderer` | Template Renderer. |
| `tileSpacing` | Tile Spacing. |
| `tileVertically` | Tile Vertically. |
| `true` | True. |
