# ImpulseZone

Зона импульса, которая применяет импульс к объектам при входе в триггер. Идеально подходит для создания зон запуска, пружин, ускорителей и других эффектов.

## Возможности

- Применение импульса при входе объекта в зону (триггер)
- Различные направления импульса
- Фильтрация по слоям и тегам
- Опциональное добавление Rigidbody
- Одноразовое или многократное срабатывание
- Задержка между срабатываниями (cooldown)
- События для интеграции

## Настройки

### Настройки импульса

- **Impulse Force**: Сила импульса
- **Direction**: Направление импульса
  - `AwayFromCenter` - от центра зоны
  - `TowardsCenter` - к центру зоны
  - `TransformForward` - по направлению Transform.forward
  - `Custom` - кастомное направление
- **Custom Direction**: Кастомное направление (для режима Custom)

### Фильтрация

- **Affected Layers**: Слои объектов, на которые будет воздействовать импульс
- **Required Tag**: Тег объектов (пусто = игнорировать фильтр по тегу)

### Опции

- **Add Rigidbody If Needed**: Автоматически добавлять Rigidbody на объекты без физики
- **One Time Only**: Одноразовое срабатывание (каждый объект может получить импульс только один раз)
- **Cooldown**: Задержка между срабатываниями для одного объекта

## События

- **OnObjectEntered**: Вызывается при входе объекта в зону (GameObject)
- **OnImpulseApplied**: Вызывается при применении импульса (GameObject)

## API

### Методы

```csharp
// Применить импульс к объекту вручную
void ApplyImpulseToObject(GameObject target)

// Установить силу импульса
void SetImpulseForce(float newForce)

// Очистить список обработанных объектов
void ClearProcessedObjects()
```

## Требования

- Компонент требует наличия Collider на том же GameObject
- Collider автоматически становится триггером при добавлении компонента
- Поддерживаются: BoxCollider, SphereCollider, CapsuleCollider

## Примеры использования

### Зона запуска объектов

1. Создайте GameObject с BoxCollider
2. Добавьте компонент `ImpulseZone`
3. Установите `Direction` на `AwayFromCenter`
4. Настройте `Impulse Force`
5. Настройте фильтрацию по слоям

### Пружина (отталкивание вверх)

1. Создайте GameObject с BoxCollider
2. Добавьте компонент `ImpulseZone`
3. Установите `Direction` на `Custom`
4. Установите `Custom Direction` на `(0, 1, 0)`
5. Настройте силу импульса

### Ускоритель (по направлению)

1. Создайте GameObject с Collider
2. Добавьте компонент `ImpulseZone`
3. Установите `Direction` на `TransformForward`
4. Поверните Transform в нужном направлении
5. Настройте силу импульса

### Одноразовая зона

1. Включите `One Time Only`
2. Каждый объект получит импульс только один раз
3. Используйте `ClearProcessedObjects()` для сброса

### Зона с задержкой

1. Установите `Cooldown` (например, 1 секунда)
2. Объект может получить импульс снова через указанное время
3. Полезно для зон, которые должны срабатывать периодически

## Визуализация

В режиме редактирования (при выборе объекта) отображается:
- Желтая полупрозрачная форма - границы зоны импульса
- Форма соответствует типу Collider (Box, Sphere, Capsule)

## Примечания

- Компонент работает с 3D триггерами
- Для объектов без Rigidbody можно включить `Add Rigidbody If Needed`
- Импульс применяется через `AddForce` с `ForceMode.Impulse`
- При `One Time Only` объекты запоминаются до вызова `ClearProcessedObjects()`
