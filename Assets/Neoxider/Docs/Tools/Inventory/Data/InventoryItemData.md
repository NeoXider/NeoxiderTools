# InventoryItemData

**Назначение:** ScriptableObject — определение предмета инвентаря (id, название, иконка, maxStack, категория).

## Подключение

- Создать: **Create → Neoxider → Inventory → Item Data**.

## Основные настройки (Inspector)

| Поле | Описание |
|------|----------|
| `Category` | Category. |
| `Description` | Description. |
| `DisplayName` | Display Name. |
| `Icon` | Icon. |
| `ItemId` | Item Id. |
| `MaxStack` | Max Stack. |
| `SupportsInstanceState` | Supports Instance State. |
| `WorldDropPrefab` | World Drop Prefab. |
