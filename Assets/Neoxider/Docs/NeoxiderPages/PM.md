# PM (Page Manager)

**Что это:** Синглтон управления UI-страницами в NeoxiderPages: включение/выключение по PageId, переключение, возврат назад, интеграция с GM (Win/Lose/End).

**Как использовать:** см. разделы ниже.

---


Синглтон управления UI-страницами в NeoxiderPages: включение/выключение по PageId, переключение, возврат назад, интеграция с GM (Win/Lose/End).

**Добавить:** GameObject → Neoxider → Pages → PM.

**Иерархия:** все **`UIPage`**, которыми управляет этот **PM**, должны находиться на **дочерних** GameObject относительно объекта с **PM** (включая неактивные объекты в этом поддереве). Авто-поиск страниц (`FindAllScenePages` / сброс списка в редакторе) обходит только это поддерево — страницы вне иерархии PM не регистрируются.

## Основное

- Страницы регистрируются по **PageId**. **UIPage** вешается на каждый экран.
- **Open(PageId)** — открыть страницу (эксклюзивно или как popup).
- **Back()** — вернуться на предыдущую страницу.
- **GM Integration** — при смене состояния GM открывать Win/Lose/End страницы.
