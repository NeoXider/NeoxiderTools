# NpcTargetFinder

**Назначение:** Компонент поиска цели для NPC (по дистанции, тегу, слою).

## Подключение

- Добавить: **Add Component → Neoxider → NPC → Target Finder**.

## Основные настройки (Inspector)

| Поле | Описание |
|------|----------|
| `_targetName` | Target Name. |
| `_targetTag` | Target Tag. |
| `false` | False. |
| `true` | True. |
