# InventorySlotGridView

**What it is:** UI grid for one `InventoryComponent` in **Slot Grid** mode; optional two-step click transfer via `InventoryTransferService`. Path: `Scripts/Tools/Inventory/UI/InventorySlotGridView.cs`.

**How to use:** Bind **Inventory** (slot-grid). Use **Slot Prefab** + **Slots Root**, or **Manual Slots**. Enable **Enable Click Transfer** for select-source then click-target between two grids.

Canonical field tables and setup examples: [InventorySlotGridView.md](../../../Docs/Tools/Inventory/InventorySlotGridView.md) (Russian).
