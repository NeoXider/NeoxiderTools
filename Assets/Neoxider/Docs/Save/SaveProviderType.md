# SaveProviderType

**Назначение:** Enum для выбора типа бэкенда сохранения.

## Значения

| Значение | Описание |
|----------|----------|
| `PlayerPrefs` | Сохранение через Unity `PlayerPrefs` (по умолчанию). |
| `File` | Сохранение в JSON-файл. |
