# Физика (Physics)

Этот раздел содержит интересные физические компоненты для создания различных эффектов и взаимодействий с физикой Unity.

## Компоненты

- [ExplosiveForce](./ExplosiveForce.md) - Взрывная сила, которая толкает объекты в радиусе
- [ImpulseZone](./ImpulseZone.md) - Зона импульса, применяющая импульс при входе объекта
- [MagneticField](./MagneticField.md) - Магнитное поле для притяжения/отталкивания объектов
  - Поддерживает притяжение к себе, к цели, к точке, отталкивание и переключение режимов

## Общие возможности

Все компоненты поддерживают:

- Фильтрацию по слоям (LayerMask)
- Опциональное добавление Rigidbody на объекты без физики
- События UnityEvent для интеграции с другими системами
- Визуализацию в редакторе через Gizmos
- Публичный API для программного управления

## Быстрый старт

### ExplosiveForce - Взрывная бочка

1. Создайте GameObject с компонентом `ExplosiveForce`
2. Настройте радиус и силу взрыва
3. Выберите режим активации (OnStart, Delayed, Manual)
4. Настройте фильтрацию по слоям
5. При необходимости включите `addRigidbodyIfNeeded`

### ImpulseZone - Зона импульса

1. Создайте GameObject с Collider (Box, Sphere, Capsule)
2. Добавьте компонент `ImpulseZone`
3. Collider автоматически станет триггером
4. Настройте направление и силу импульса
5. Настройте фильтрацию по слоям и тегам

### MagneticField - Магнитное поле

1. Создайте GameObject с компонентом `MagneticField`
2. Настройте радиус и силу поля
3. Выберите режим (Attract, Repel, Toggle, ToTarget, ToPoint)
4. Настройте фильтрацию по слоям
5. Поле будет работать в FixedUpdate для стабильной физики

```csharp
// MagneticField в режиме Attract
```
