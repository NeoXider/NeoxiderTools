# GridShapeMask

**Назначение:** Маска формы для GridSystem — определяет активные ячейки сетки.

## Подключение

- Создать: **Create → Neoxider → Grid → Shape Mask**.

## Основные настройки (Inspector)

| Поле | Описание |
|------|----------|
| `Cells` | Cells. |
| `GridShapeMaskMode` | Grid Shape Mask Mode. |
| `Mode` | Mode. |
| `_cells` | Cells. |
| `_mode` | Mode. |
