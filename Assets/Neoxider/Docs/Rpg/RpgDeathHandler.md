# RpgDeathHandler

**Назначение:** Обработчик смерти `RpgCombatant`, позволяющий гибко настроить логику (анимации, удаление объекта, спавн лута) при нулевом здоровье.

## Подключение

- Работает в связке с `RpgCombatant`.
