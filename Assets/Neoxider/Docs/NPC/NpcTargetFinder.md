# NpcTargetFinder

**Назначение:** Документация сгенерирована автоматически.

## Подключение

- Добавьте компонент через меню Unity.

## Основные настройки (Inspector)

| Поле | Описание |
|------|----------|
| `_targetName` | Target Name. |
| `_targetTag` | Target Tag. |
| `false` | False. |
| `true` | True. |
